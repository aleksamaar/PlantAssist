# ☁️ Переезд в облако (PythonAnywhere)

Цель: приложение работает всегда, без компьютера, по адресу
`https://ТВОЙ_ЛОГИН.pythonanywhere.com` — с настоящим HTTPS, чтобы на телефон
поставилась иконка.

---

## 1. Регистрация (5 мин)

1. Открой **pythonanywhere.com** → **Pricing & signup** → **Create a Beginner account**
   (бесплатно, карта не нужна).
2. Придумай логин — он станет частью адреса: `https://ЛОГИН.pythonanywhere.com`.
3. Подтверди почту.

## 2. Загрузка кода (5 мин)

1. В меню сверху → **Consoles** → **Bash**.
2. Введи команду:
   ```bash
   git clone https://github.com/aleksamaar/PlantAssist.git
   ```
3. Установи Flask:
   ```bash
   pip3 install --user flask
   ```

## 3. Настройка веб-приложения (10 мин)

1. Меню → **Web** → **Add a new web app** → **Next**.
2. Выбери **Manual configuration** (не «Flask»!) → **Python 3.10** → **Next**.
3. На странице Web заполни:
   - **Source code:** `/home/ЛОГИН/PlantAssist`
   - **Working directory:** `/home/ЛОГИН/PlantAssist`
4. Нажми на ссылку **WSGI configuration file** и замени всё содержимое на:

   ```python
   import sys
   import os

   path = '/home/ЛОГИН/PlantAssist'      # ← подставь свой логин
   if path not in sys.path:
       sys.path.insert(0, path)

   os.environ['PLANTASSIST_PASSWORD'] = 'придумай-пароль'       # ← пароль на вход
   os.environ['PLANTASSIST_SECRET'] = 'длинная-случайная-строка' # ← любые ~40 символов

   from app import app as application
   ```
   Сохрани (**Save**).
5. Вернись на вкладку **Web** → большая зелёная кнопка **Reload**.

## 4. Первый вход и перенос данных (5 мин)

1. Открой `https://ЛОГИН.pythonanywhere.com` → появится страница входа → введи пароль.
2. Приложение откроется **пустым** — это нормально, данные ещё здесь, на компе.
3. Перенос: вкладка **Статистика** → внизу **💾 Данные и бэкап** →
   **♻️ Восстановить из бэкапа** → выбери свежий ZIP из папки `backups/`.
4. Готово — все 36 растений, история и фото на месте.

> Свежий бэкап делается запуском `backup.py` на компе (пока сервер запущен локально).

## 5. Иконка на телефоне (2 мин)

1. На телефоне открой в **Chrome**: `https://ЛОГИН.pythonanywhere.com` → войди.
2. Меню Chrome (⋮) → **«Установить приложение»**.
3. Готово — иконка 🌿 в списке приложений. **Никаких сертификатов.**

## 6. Автобэкап в Яндекс Диск (3 мин)

1. Открой `backup.bat` в блокноте и впиши свои данные:
   ```bat
   set PLANTASSIST_URL=https://ЛОГИН.pythonanywhere.com
   set PLANTASSIST_PASSWORD=твой-пароль
   ```
2. Проверь — запусти двойным кликом. Должно написать `OK: ... МБ, фото: N`.
3. Автоматически раз в день:
   - **Пуск** → **Планировщик заданий** → **Создать простую задачу**
   - Имя: `PlantAssist бэкап` → Триггер: **Ежедневно** → Действие: **Запустить программу**
   - Программа: путь к `backup.bat`
4. Бэкапы копятся в `backups/` внутри Яндекс Диска (хранятся 30 последних).

---

## Важно знать

- **Раз в 3 месяца** PythonAnywhere просит зайти и нажать кнопку продления на вкладке
  **Web** (иначе приложение замораживают). Даже если забудешь — данные не потеряны,
  они есть в Яндекс Диске, и восстановить можно за 15 минут.
- **Облако теперь главное.** И телефон, и компьютер работают с ним. Локальная папка
  остаётся как бэкап/разработка.
- **Обновление кода:** после изменений на компе — `git push`, затем на PythonAnywhere
  в Bash: `cd PlantAssist && git pull`, и кнопка **Reload** на вкладке Web.
- **Локально пароль не спрашивается** — он включается только если задана переменная
  `PLANTASSIST_PASSWORD` (она есть только на хостинге).
